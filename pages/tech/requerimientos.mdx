# Requerimientos Técnicos

## Requerimientos del Sistema

### Aplicativo Web

- **Compatibilidad de Navegadores**:
  El aplicativo web es compatible con los siguientes navegadores:

  - Google Chrome (versión 90 o superior)
  - Mozilla Firefox (versión 80 o superior)
  - Microsoft Edge (versión 90 o superior)
  - Safari (versión 14 o superior)
  - Opera (versión 75 o superior)

- **Requisitos del Sistema**:
  - **Conexión a Internet**: Requerido para acceder a la plataforma en línea.
  - **Resolución de Pantalla**: Recomendado 1024x768 píxeles o superior.
  - **Navegadores recomendados**: Google Chrome, Mozilla Firefox o Microsoft Edge para mejor rendimiento.

### Aplicativo Móvil

- **Sistema Operativo**:
  El aplicativo móvil está diseñado para funcionar exclusivamente en el sistema operativo Android.

  - **Requisitos de Android**:

    - **Versión mínima de Android**: Android 5.0 Lollipop o superior.
    - **Espacio de almacenamiento**: Al menos 100 MB disponibles.
    - **Conexión a Internet**: Requerida para el acceso a productos y recomendaciones.

  - **Compatibilidad con Dispositivos**:
    - Compatible con smartphones y tabletas Android.

## Requerimientos de Software

### Backend

- **Servidor**:
  - **Lenguaje de programación**: Python(Flask, Scrapy y Selenium).
  - **Base de Datos**: MongoDB.
  - **Servicios adicionales**: API RESTful para interacción con el frontend y otros servicios.

### Frontend

- **Framework**:
  - **Frontend Web**: React.js.
  - **Frontend Móvil**: PWA

### Otros

- **Servicios en la Nube**:
  - **Hosting**: Cloud Run de Google Cloud (Backend) y Vercel (Frontend).

## Consideraciones Adicionales

- **Seguridad**: El sistema implementa HTTPS para asegurar la comunicación entre el frontend y el backend.
